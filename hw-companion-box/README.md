# Companion Box

Auxiliary unit for the QO-100 transverter, providing:

- 13.8V to 28V voltage boost for powering the transverter from a standard 13.8V shack supply
- PTT combiner: mixes PTT signal with RF on the coax line to the transverter

> **Note:** unplugging the PTT line triggers an audible alert from the built-in buzzer.

See [schematics](sch_companion_box.pdf) for details.
