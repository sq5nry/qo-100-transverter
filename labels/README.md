# Labels

Label artwork for the QO-100 transverter enclosures (front panel, rear panel, outdoor versions).

## PZK Logo

The label artwork includes the logo of [PZK — Polski Związek Krótkofalowców](https://www.pzk.org.pl/)
(Polish Amateur Radio Union). The PZK logo is the property of PZK and is used in this project
with PZK's permission.

Anyone reproducing these labels must either obtain their own permission from PZK before using
their logo, or replace it with their own club or personal artwork.
